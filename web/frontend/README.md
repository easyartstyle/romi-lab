# Frontend

Web UI для клиента и администратора, который должен сохранить текущий UX desktop-программы.

## Что уже есть

- логин
- список проектов
- открытие проекта
- backend-driven dashboard
- импорт рекламных и CRM JSON в проект

## Что нужно для локального запуска

1. Установленный Node.js LTS
2. `npm install`
3. Запуск Next.js dev server

## Быстрый запуск

```powershell
cd Q:\CODEX\web\frontend
npm install
npm run dev
```

По умолчанию frontend ожидает backend на:

```text
http://127.0.0.1:8000/api
```

Если нужно другое значение, можно задать переменную:

```text
NEXT_PUBLIC_API_BASE_URL
```

## UX-принцип

Это не новый отдельный интерфейс, а web-версия текущего приложения:

- те же KPI-карточки;
- те же вкладки и фильтры;
- тот же порядок работы с проектами;
- тот же экран аналитики и планирования.
